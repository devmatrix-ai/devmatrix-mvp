# Frontend CSS

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend CSS.

## When to use this skill:

- When writing or editing CSS files (e.g., `*.css`, `*.scss`, `*.module.css`) or styled-components
- When applying Tailwind CSS utility classes, BEM naming conventions, or other CSS methodologies
- When establishing or using design tokens for colors, spacing, typography, and other design values
- When creating theme configurations or design system documentation
- When optimizing CSS for production through purging unused styles or tree-shaking
- When working with CSS frameworks like Tailwind CSS, Bootstrap, Material-UI, or Chakra UI
- When avoiding framework style overrides and working with the framework's patterns
- When adding styles to React components, Vue components, or other frontend files
- When ensuring CSS consistency and maintainability across the codebase
