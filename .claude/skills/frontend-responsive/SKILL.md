# Frontend Responsive

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend responsive.

## When to use this skill:

- When creating or editing component files that include responsive layouts or breakpoints
- When implementing mobile-first design patterns starting with mobile layouts
- When defining or using responsive breakpoints (mobile, tablet, desktop) with Tailwind or media queries
- When using relative units (rem, em, %) for responsive typography and spacing
- When creating fluid layouts with flexbox or grid that adapt to screen size
- When ensuring touch-friendly tap targets (minimum 44x44px) for mobile interfaces
- When optimizing images and assets for mobile network conditions
- When testing components across multiple screen sizes and devices
- When prioritizing content display for smaller screens through thoughtful layout decisions
